## Unified Virtual Environment

Prior to 0.40.0, each tool had its own virtual environment:
- `~/.maxinger15-music/mastering-env/` — mastering tools
- `~/.maxinger15-music/promotion-env/` — promo video tools
- `~/.maxinger15-music/cloud-env/` — cloud upload tools

These are now consolidated into a single `~/.maxinger15-music/venv/` managed by the setup skill. The old directories can be safely removed.

If auto-cleanup didn't run, use the `cleanup_legacy_venvs` MCP tool or delete manually.

If you haven't run `$maxinger15-music:setup` recently, do so to create the unified venv with all dependencies.
